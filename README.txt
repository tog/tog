tog
============
tog is the gem that boostraps the process of _togifing_ your app/site.

Install
-------

```
sudo gem install tog
```

Or from source (you will need Hoe >=1.5.3 and mislav-will_paginate ~> 2.3.2):

```
git clone git@github.com:tog/tog.git  
cd tog
rake install_gem
```
